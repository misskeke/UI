三角形

给 <i> 标签添加 .triangle-upward 类可以模拟一个向上的三角形。三角形的方向、大小和颜色可以通过混合函数里的变量重新定义。

样式

向上的三角形 向下的三角形 向左的三角形 向右的三角形

html代码

                    
                        <!--向上的三角形-->
                        <i class="triangle-upward"></i>

                        <!--向下的三角形-->
                        <i class="triangle-downward"></i>

                        <!--向左的三角形-->
                        <i class="triangle-leftward"></i>

                        <!--向右的三角形-->
                        <i class="triangle-rightward"></i>

图标

给 <i> 标签添加 .icon 类设置字体图标基础样式。

添加 .bg- 类可以为图标添加不同的背景颜色，如添加 .bg-green 可以为图标添加一个绿色的背景。

可用的背景色包括： .bg-green、.bg-blue、.bg-red、.bg-yellow、.bg-purple、.bg-black、.bg-gray。

添加 .color-white 类可以让字体图标的颜色显示为白色。

样式

普通图标 可操作的图标 禁止操作的图标

html结构

                    
                        <!--普通图标-->
                        <i class="icon icon-reply" title="回复"></i>

                        <!--可操作的图标-->
                        <i class="icon icon-primary icon-reply bg-green" title="回复"></i>
                        <i class="icon icon-primary icon-reply bg-blue" title="回复"></i>

                        <!--禁止操作的图标-->
                        <i class="icon icon-primary icon-reply bg-gray disabled" title="回复"></i>

按钮

给 <button> 标签添加 .btn-normal 或 .btn-big 类设置按钮基础样式。 .btn-normal 和 .btn-big 分别定义了两种不同大小的按钮。

添加 .bg- 类可以为图标添加不同的背景颜色，同时按钮的文字显示为白色。如添加 .bg-green 可以为按钮添加一个绿色的背景。

可用的背景色包括： .bg-green、.bg-blue、.bg-red、.bg-yellow、.bg-purple、.bg-black、.bg-gray。

样式

html结构

                    
                        <!-- 幽灵按钮 -->
                        <button class="btn-spirit">幽灵按钮</button>

                        <!-- 禁止按钮 -->
                        <button class="btn-disabled">禁止按钮</button>

                        <!-- 基础按钮 -->
                        <button class="btn-normal bg-green" type="button">绿色按钮</button>
                        <button class="btn-normal bg-blue" type="button">蓝色按钮</button>

                        <!-- 大按钮 -->
                        <button class="btn-big bg-green" type="button">绿色按钮</button>
                        <button class="btn-big bg-blue" type="button">蓝色按钮</button>

按钮使用

当有两个或以上的按钮时，在按钮组的外层添加一个类名为 .btn-box 的 div，可以使按钮 水平居中 显示，且按钮和按钮之间的间距为 20px 。

html结构

                    
                        <div class="btn-box">
                            <button class="btn-normal bg-green">结算</button>
                            <button class="btn-normal bg-green">筛选</button>
                            <button class="btn-normal bg-blue">导出</button>
                        </div>
